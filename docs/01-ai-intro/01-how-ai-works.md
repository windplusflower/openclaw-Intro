# AI 如何工作

> 理解大模型如何"思考"

---

## 🤖 什么是大模型

**大模型**（Large Language Model，LLM）是一个经过大量文本训练的 AI 系统。

### 通俗理解

想象一个读过互联网上几乎所有文字的人：
- 📚 读过维基百科、书籍、论坛、新闻
- 💬 见过无数对话和问答
- 🧠 学会了语言的规律和模式

**但它并不真正"理解"**，只是预测下一个字应该是什么。

---

## 🧠 AI 如何"思考"

### 工作原理

```
输入 → 分词 → 预测 → 输出
```

1. **输入**：你发送一段文字
2. **分词**：AI 把文字拆成 Token（词元）
3. **预测**：AI 预测下一个 Token 应该是什么
4. **输出**：重复预测，直到生成完整回答

### 可视化示例

```
用户输入：今天天气真

AI 预测：
- "好" (80% 概率)
- "不错" (15% 概率)
- "糟糕" (5% 概率)

AI 输出：今天天气真好
```

## ⚡ AI 的特点

### ✅ 优点

| 特点 | 说明 |
|------|------|
| **知识广博** | 读过海量文本，知道很多事 |
| **反应快速** | 几秒钟就能生成回答 |
| **多才多艺** | 写作、编程、分析都能做 |
| **不知疲倦** | 随时待命 |

### ❌ 局限

| 局限 | 说明 |
|------|------|
| **没有真实理解** | 只是预测文字，不理解含义 |
| **会胡说八道** | 可能生成看似合理但错误的内容 |
| **知识有截止** | 训练数据有截止日期 |
| **没有记忆** | 每次对话都是新的开始（除非在同一个会话中） |

## 💡 关键概念

### 1. AI 不是搜索引擎

- 🔍 **搜索引擎**：查找已有的信息
- 🤖 **AI**：生成新的文字（可能不准确）

### 2. AI 没有记忆

- 每次对话都是新的开始
- 在同一个会话中才有"上下文记忆"
- 关闭会话后，AI 会"忘记"一切

### 3. AI 不会思考

- 只是按概率预测下一个字
- 没有意识、没有情感、没有目的
- 看起来像思考，实际是数学计算

## 📝 实践示例

### 示例 1：对比不同 AI

用同样的问题问 3 个不同的 AI：

```
问题：中国的首都是哪里？
```

观察：
- [ ] 回答是否一致？
- [ ] 哪个 AI 回答更详细？
- [ ] 有没有 AI 胡说？

### 示例 2：测试 AI 的局限

问 AI 一个 2025 年之后发生的事：

```
问题：2026 年 3 月的诺贝尔奖得主是谁？
```

观察：
- [ ] AI 是否承认不知道？
- [ ] AI 是否胡编乱造？
- [ ] 如何验证 AI 的回答？

## ⚠️ 重要提醒

### AI 会胡说

AI 可能生成**看似合理但完全错误**的内容，特别是：

- 📅 最新事件（超出训练数据）
- 🔢 具体数字、日期、引用
- 👤 人物传记、公司信息
- 📜 法律、医疗建议

### 始终验证

对于重要信息：
1. ✅ 用搜索引擎二次确认
2. ✅ 查看官方来源
3. ✅ 咨询专业人士
