## 💻 双平台支持

本文档同时支持 **macOS** 和 **Windows** 用户：

- 🍎 **macOS 用户**：使用 Homebrew 安装，终端命令以 `brew` 开头
- 🪟 **Windows 用户**：使用 Winget 安装，终端使用 PowerShell

每个文档都会标注双平台命令对照。
